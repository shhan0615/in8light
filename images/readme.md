# 공유 이미지 안내

## share-image.png

이 디렉토리에는 카카오톡 공유 시 표시되는 `share-image.png` 파일이 포함되어 있습니다.

### 현재 상태
✅ 공유 이미지가 이미 포함되어 있습니다!

### 이미지 사양
- **파일명**: share-image.png
- **크기**: 1406x835px (46KB)
- **형식**: PNG
- **내용**: 
  - 🌿 in8 로고 (초록색 나뭇잎 + 파란색 텍스트)
  - in8 서비스 설명
  - 카카오 간편 로그인 버튼

### 사용 방법

#### 1. 로컬 테스트 시
로컬 환경(file://)에서는 카카오톡 공유 이미지가 표시되지 않습니다.
웹 서버를 통해 테스트해야 이미지가 표시됩니다.

#### 2. 웹 서버 배포 후
웹 서버에 배포하면 자동으로 이미지가 표시됩니다.
- 이미지 경로: `https://yourdomain.com/images/share-image.png`
- 코드가 자동으로 현재 도메인의 이미지를 사용합니다

#### 3. 이미지 교체 시
다른 이미지를 사용하고 싶다면:
1. 새 이미지를 `share-image.png`로 저장 (기존 파일 덮어쓰기)
2. 권장 크기: 800x400px 이상
3. 형식: PNG 또는 JPG

### 테스트 방법
```bash
# 로컬 웹 서버 실행 (예시)
python -m http.server 8000
# 또는
npx http-server

# 브라우저에서 접속
http://localhost:8000
```

### 참고
- 카카오톡 공유는 외부에서 접근 가능한 URL이 필요합니다
- 로컬 파일(file://)에서는 이미지가 표시되지 않습니다
- 웹 서버에 배포하면 정상적으로 이미지가 표시됩니다

## PWA 아이콘 (v1.8.3 추가)

### 포함된 파일
- `icon.svg` - 벡터 아이콘 (원본)
- `icon-192.png` - 192x192 PNG 아이콘 (필요시 생성)
- `icon-512.png` - 512x512 PNG 아이콘 (필요시 생성)

### PWA 아이콘 생성 방법

icon.svg 파일을 사용하여 다음 크기의 PNG 파일을 생성해야 합니다:

1. **icon-192.png** (192x192)
2. **icon-512.png** (512x512)

#### 온라인 변환 도구 사용:
1. https://cloudconvert.com/svg-to-png
2. icon.svg 업로드
3. 원하는 크기로 변환 (192x192, 512x512)
4. 다운로드 후 images 폴더에 저장

#### ImageMagick 사용:
```bash
convert icon.svg -resize 192x192 icon-192.png
convert icon.svg -resize 512x512 icon-512.png
```

### 아이콘 디자인
- 배경: 보라색 그라데이션 (#667eea → #764ba2)
- 중앙: 8개 잎을 가진 녹색 나뭇잎 로고
- 하단: "in8" 흰색 텍스트

### 주의사항
- PWA 아이콘은 정사각형이어야 합니다 (512x512)
- 투명 배경보다는 불투명 배경 권장
- 아이콘은 다양한 배경색에서 잘 보여야 합니다
- 홈 화면에 추가 시 이 아이콘이 사용됩니다
