## 📋 주요 변경사항

### ⚕️ 섭생표 기반 체질별 정보 완전 재작성 (중요!)

**v0.8.91 개선:**

- 모든 음식 정보가 정확함 ✅

## 📊 체질별 수정 내용 (섭생표 기준)

### 🌳 목체질 (목양·목음) - 간담의 기운

**섭생표 데이터 분석:**
- OO(매우 좋음): 쇠고기, 돼지고기(목음), 뿌리채소
- XX(매우 나쁨): 모든 바다생선, 조개류, 해산물

**수정 내용:**
```
✅ 좋은 음식:
  - 쇠고기 (목양·목음 모두 OO)
  - 돼지고기 (목음 OO, 목양 Δ)
  - 뿌리채소 (무, 당근, 연근, 우엉, 감자) OO
  - 계란 (O)
  - 온유제품 (O)

⚠️ 피해야 할 음식:
  - 모든 생선 (XX)
  - 모든 해산물 (조개, 새우, 게, 굴) XX
  - 푸른잎채소 과다 (Δ)
  - 찬 음식 (냉유제품)
```

**핵심:** 목체질은 **육식+뿌리채소** 중심, **생선·해산물 절대 금물**

### 🏔️ 금체질 (금양·금음) - 폐대장의 기운

**섭생표 데이터 분석:**
- OO(매우 좋음): 바다생선, 조개류, 푸른잎채소
- XX(매우 나쁨): 모든 육류, 뿌리채소

**수정 내용:**
```
✅ 좋은 음식:
  - 바다생선 (흰살생선, 붉은살생선) OO
  - 조개류·해산물 (조개, 새우, 게) OO
  - 푸른잎채소 (배추, 양배추, 상추) OO
  - 계란흰자 (O)
  - 쌀·곡류 (O)

⚠️ 피해야 할 음식:
  - 쇠고기 (XX)
  - 돼지고기 (XX)
  - 닭고기 (XX)
  - 뿌리채소 (무, 당근, 감자) XX
  - 유제품 (우유, 버터, 치즈) XX
```

**핵심:** 금체질은 **해식+채식** 중심, **모든 육류 절대 금물**

---

### ⛰️ 토체질 (토양·토음) - 비위의 기운

**섭생표 데이터 분석:**
- OO(매우 좋음): 쇠고기(토양), 돼지고기
- O(좋음): 바다생선, 조개류, 푸른잎채소, 냉유제품
- XX(매우 나쁨): 닭고기, 염소고기

**수정 내용:**
```
✅ 좋은 음식:
  - 쇠고기 (토양 OO, 토음 Δ)
  - 돼지고기 (토양·토음 모두 OO)
  - 바다생선 (O)
  - 조개류·해산물 (O)
  - 푸른잎채소 (O)
  - 냉유제품 (찬 우유, 요거트) O

⚠️ 피해야 할 음식:
  - 닭고기·오리고기 (XX)
  - 염소고기 (XX)
  - 일부 뿌리채소 (감자, 고구마, 마) X
  - 온유제품 (따뜻한 우유) Δ/X
```

**핵심:** 토체질은 **쇠고기·돼지고기+해산물+채식**, **닭고기 절대 금물**

### 💧 수체질 (수양·수음) - 신방광의 기운

**섭생표 데이터 분석:**
- OO(매우 좋음): 닭고기, 염소고기, 쇠고기(수음)
- XX(매우 나쁨): 돼지고기(수양), 조개류, 굴

**수정 내용:**
```
✅ 좋은 음식:
  - 닭고기·오리고기 (OO)
  - 염소고기 (OO)
  - 쇠고기 (수음 OO, 수양 Δ)
  - 뿌리채소 (무, 당근, 감자, 고구마, 마)
  - 계란 (O)
  - 푸른잎채소 (O)

⚠️ 피해야 할 음식:
  - 돼지고기 (수양 XX, 수음 XX)
  - 조개류 (조개, 굴) X/XX
  - 새우·게 (갑각류) X/XX
  - 일부 생선 (붉은살 생선) Δ
  - 냉유제품 (찬 우유) X
```

**핵심:** 수체질은 **닭고기·염소고기+뿌리채소**, **돼지고기·조개류 금물**
